| الآية | التفسير |
|-------|---------|
| **إِذَا وَقَعَتِ الْوَاقِعَةُ** (1) | إذا حدثت القيامة، وهي واقعة حقيقية لا محالة. |
| **لَيْسَ لِوَقْعَتِهَا كَاذِبَةٌ** (2) | لا أحد يستطيع إنكار وقوعها أو التهرب من حقيقتها. |
| **خَافِضَةٌ رَّافِعَةٌ** (3) | تخفض أقوامًا وترفع أقوامًا، أي تهين بعضهم وتعز آخرين وفق أعمالهم. |
| **إِذَا رُجَّتِ الْأَرْضُ رَجًّا** (4) | حين تهتز الأرض اهتزازًا شديدًا عند وقوع القيامة. |
| **وَبُسَّتِ الْجِبَالُ بَسًّا** (5) | وتتحطم الجبال وتتناثر. |
| **فَكَانَتْ هَبَاءً مُّنبَثًّا** (6) | وتصبح الجبال كالغبار المتناثر في الهواء. |
| **وَكُنتُمْ أَزْوَاجًا ثَلَاثَةً** (7) | ويُقسم الناس يوم القيامة إلى ثلاث مجموعات أو أصناف. |
| **فَأَصْحَابُ الْمَيْمَنَةِ مَا أَصْحَابُ الْمَيْمَنَةِ** (8) | أصحاب اليمين هم من يدخلون الجنة، وهم في مقام محمود. |
| **وَأَصْحَابُ الْمَشْأَمَةِ مَا أَصْحَابُ الْمَشْأَمَةِ** (9) | وأصحاب الشمال هم من يذهبون إلى النار، وهم في شقاء. |
| **وَالسَّابِقُونَ السَّابِقُونَ** (10) | والسابقون هم الذين تفوقوا في الخير، وهم الأوائل في طاعة الله. |
| **أُوْلَئِكَ الْمُقَرَّبُونَ** (11) | هؤلاء هم الذين قربهم الله إليه، لهم مكانة خاصة عنده. |
| **فِي جَنَّاتِ النَّعِيمِ** (12) | سيكونون في الجنان، يعيشون في نعيم دائم. |
| **ثُلَّةٌ مِّنَ الْأَوَّلِينَ** (13) | جماعة كبيرة منهم من السابقين الأولين. |
| **وَقَلِيلٌ مِّنَ الْآخِرِينَ** (14) | وقليل منهم من الأمم المتأخرة. |
| **عَلَى سُرُرٍ مَّوْضُونَةٍ** (15) | يجلسون على أسِرَّة مصنوعة من الذهب والجواهر الثمينة. |
| **مُّتَّكِئِينَ عَلَيْهَا مُتَقَابِلِينَ** (16) | يجلسون متقابلين على تلك الأسرّة في سعادة وراحة. |
| **يَطُوفُ عَلَيْهِمْ وِلْدَانٌ مُّخَلَّدُونَ** (17) | يخدم أهل الجنة غلمان شباب دائمون لا يشيخون. |
| **بِأَكْوَابٍ وَأَبَارِيقَ وَكَأْسٍ مِّن مَّعِينٍ** (18) | يحملون لهم أكوابًا وأباريقًا وكؤوسًا مملوءة بشراب خالص من عين جارية. |
| **لَا يُصَدَّعُونَ عَنْهَا وَلَا يُنزِفُونَ** (19) | لا يصيبهم من هذا الشراب صداع ولا يفقدون عقولهم منه، أي أنه شراب لذيذ بلا ضرر. |
| **وَفَاكِهَةٍ مِّمَّا يَتَخَيَّرُونَ** (20) | يُقدّم لهم فاكهة من كل نوع يختارونها ويرغبون فيها. |
| **وَلَحْمِ طَيْرٍ مِّمَّا يَشْتَهُونَ** (21) | ويقدم لهم لحم طيور مما يرغبون ويشتهون. |
| **وَحُورٌ عِينٌ** (22) | ويكونون مع زوجات من الحور العين، نساء الجنة الجميلات. |
| **كَأَمْثَالِ اللُّؤْلُؤِ الْمَكْنُونِ** (23) | الحور العين مثل اللؤلؤ المحفوظ في الصدفة، دليل على جمالهنّ وصفائهنّ. |
| **جَزَاءً بِمَا كَانُوا يَعْمَلُونَ** (24) | هذا الجزاء هو مقابل أعمالهم الصالحة التي قاموا بها في الدنيا. |
| **لَا يَسْمَعُونَ فِيهَا لَغْوًا وَلَا تَأْثِيمًا** (25) | لا يسمعون في الجنة كلامًا باطلًا أو مُذنبًا، بل كل ما يسمعونه هو خير وسلام. |
| **إِلَّا قِيلًا سَلَامًا سَلَامًا** (26) | يسمعون فقط كلام السلام والطمأنينة. |
| **وَأَصْحَابُ الْيَمِينِ مَا أَصْحَابُ الْيَمِينِ** (27) | أهل اليمين هم أصحاب الجنة، وقد بلغوا مكانة عظيمة. |
| **فِي سِدْرٍ مَّخْضُودٍ** (28) | يجلسون في ظلال شجر السدر الذي لا شوك فيه. |
| **وَطَلْحٍ مَّنضُودٍ** (29) | وأشجار الطلح المتراكمة ثمارها بعضها فوق بعض. |
| **وَظِلٍّ مَّمْدُودٍ** (30) | وظلال ممتدة لا تنقطع. |
| **وَمَاءٍ مَّسْكُوبٍ** (31) | وماء يجري ويُسكب دون انقطاع. |
| **وَفَاكِهَةٍ كَثِيرَةٍ** (32) | وفاكهة وفيرة لا تنتهي. |
| **لَّا مَقْطُوعَةٍ وَلَا مَمْنُوعَةٍ** (33) | هذه الفاكهة لا تنقطع في أي وقت ولا يُمنع أحد من تناولها. |
| **وَفُرُشٍ مَّرْفُوعَةٍ** (34) | وفراش مرتفع مُعَدّ لهم للراحة. |
| **إِنَّا أَنشَأْنَاهُنَّ إِنشَاءً** (35) | إن الله خلق نساء الجنة إنشاءً جديدًا غير ما كنّ عليه في الدنيا. |
| **فَجَعَلْنَاهُنَّ أَبْكَارًا** (36) | جعلهن عذارى دائمًا، أي دائمات الشباب والنقاء. |
| **عُرُبًا أَتْرَابًا** (37) | ودودات لعشاقهن متساويات في السن، متوافقات في أعمارهن. |
| **لِّأَصْحَابِ الْيَمِينِ** (38) | كل هذا لأهل اليمين الذين دخلوا الجنة. |
| **ثُلَّةٌ مِّنَ الْأَوَّلِينَ** (39) | جماعة كبيرة من السابقين الأوائل. |
| **وَثُلَّةٌ مِّنَ الْآخِرِينَ** (40) | وجماعة أخرى من المتأخرين. |
| **وَأَصْحَابُ الشِّمَالِ مَا أَصْحَابُ الشِّمَالِ** (41) | أما أهل الشمال فهم أهل النار، في حالة سيئة جدًا. |
| **فِي سَمُومٍ وَحَمِيمٍ** (42) | يعيشون في ريح حارة شديدة وفي ماء يغلي. |
| **وَظِلٍّ مِّن يَحْمُومٍ** (43) | تحت ظل دخان أسود كثيف خانق. |
| **لَا بَارِدٍ وَلَا كَرِيمٍ** (44) | هذا الظل ليس باردًا ولا مريحًا، بل عذاب مستمر. |
| **إِنَّهُمْ كَانُوا قَبْلَ ذَلِكَ مُتْرَفِينَ** (45) | هؤلاء كانوا في الدنيا مترفين، يعيشون حياة الغنى والرفاهية دون مراعاة طاعة الله. |
| **وَكَانُوا يُصِرُّونَ عَلَى الْحِنثِ الْعَظِيمِ** (46) | كانوا مستمرين في ارتكاب الذنوب العظيمة مثل الشرك والكفر. |
| **وَكَانُوا يَقُولُونَ أَئِذَا مِتْنَا وَكُنَّا تُرَابًا وَعِظَامًا أَئِنَّا لَمَبْعُوثُونَ** (47) | وكانوا يسخرون من البعث، قائلين: هل سنبعث بعد أن نصبح ترابًا وعظامًا؟ |
| **أَوَآبَاؤُنَا الْأَوَّلُونَ** (48) | وهل آباؤنا الأوائل أيضًا سيبعثون؟ كانوا مستنكرين الفكرة بشكل ساخر. |
| **قُلْ إِنَّ الْأَوَّلِينَ وَالْآخِرِينَ** (49) | قل لهم: نعم، سيبعث الله جميع الأولين والآخرين. |
| **لَمَجْمُوعُونَ إِلَى مِيقَاتِ يَوْمٍ مَّعْلُومٍ** (50) | سيُجمع الجميع في يوم محدد معروف وهو يوم القيامة. |
إليك تفسيرًا مختصرًا للآيات من 51 إلى نهاية سورة الواقعة مع توضيح الكلمات غير الدارجة:

| الآية | التفسير |
|-------|---------|
| **ثُمَّ إِنَّكُمْ أَيُّهَا الضَّالُّونَ الْمُكَذِّبُونَ** (51) | ثم أنتم أيها الضالون الذين كذبتم بالبعث والحساب. |
| **لَآكِلُونَ مِن شَجَرٍ مِّن زَقُّومٍ** (52) | سوف تأكلون من شجرة الزقوم، وهي شجرة مرّة وخبيثة تنبت في جهنم. |
| **فَمَالِئُونَ مِنْهَا الْبُطُونَ** (53) | تملؤون بطونكم من ثمارها، رغم طعمها المر وعذابها. |
| **فَشَارِبُونَ عَلَيْهِ مِنَ الْحَمِيمِ** (54) | ثم تشربون فوق ذلك من ماء شديد الغليان. |
| **فَشَارِبُونَ شُرْبَ الْهِيمِ** (55) | تشربون مثل شرب الإبل التي أصابها العطش الشديد، فلا تروى مهما شربت. |
| **هَذَا نُزُلُهُمْ يَوْمَ الدِّينِ** (56) | هذا هو ما يستقبلهم به يوم القيامة، وهو أول ما ينالونه من العذاب. |
| **نَحْنُ خَلَقْنَاكُمْ فَلَوْلَا تُصَدِّقُونَ** (57) | نحن خلقناكم، فلماذا لا تصدقون بحقيقة البعث؟ |
| **أَفَرَأَيْتُم مَّا تُمْنُونَ** (58) | هل فكرتم في النطفة التي تخرج منكم عند الإنجاب؟ |
| **أَأَنتُمْ تَخْلُقُونَهُ أَمْ نَحْنُ الْخَالِقُونَ** (59) | هل أنتم الذين تخلقون الجنين أم نحن الخالقون؟ |
| **نَحْنُ قَدَّرْنَا بَيْنَكُمُ الْمَوْتَ وَمَا نَحْنُ بِمَسْبُوقِينَ** (60) | نحن قدّرنا الموت بينكم ولا يستطيع أحدٌ أن يهرب من الموت أو يعوقه. |
| **عَلَى أَن نُّبَدِّلَ أَمْثَالَكُمْ وَنُنشِئَكُمْ فِي مَا لَا تَعْلَمُونَ** (61) | نحن قادرون على أن نستبدلكم بأناس آخرين ونخلقكم من جديد في أحوال لا تعرفونها. |
| **وَلَقَدْ عَلِمْتُمُ النَّشْأَةَ الْأُولَى فَلَوْلَا تَذَكَّرُونَ** (62) | لقد علمتم كيف خُلقتم أول مرة، فلماذا لا تتذكرون وتفكرون في البعث؟ |
| **أَفَرَأَيْتُم مَّا تَحْرُثُونَ** (63) | هل تأملتم في الزرع الذي تزرعونه؟ |
| **أَأَنتُمْ تَزْرَعُونَهُ أَمْ نَحْنُ الزَّارِعُونَ** (64) | هل أنتم الذين تجعلونه ينمو أم نحن الذين ننبت الزرع؟ |
| **لَوْ نَشَاءُ لَجَعَلْنَاهُ حُطَامًا فَظَلْتُمْ تَفَكَّهُونَ** (65) | لو شئنا لجعلنا الزرع يتلف ويصبح حطامًا، فتتحيرون وتأسفون. |
| **إِنَّا لَمُغْرَمُونَ** (66) | تقولون حينها: نحن واقعون في خسارة وهلاك. |
| **بَلْ نَحْنُ مَحْرُومُونَ** (67) | بل نحن محرومون من الرزق. |
| **أَفَرَأَيْتُمُ الْمَاءَ الَّذِي تَشْرَبُونَ** (68) | هل فكرتم في الماء الذي تشربونه؟ |
| **أَأَنتُمْ أَنزَلْتُمُوهُ مِنَ الْمُزْنِ أَمْ نَحْنُ الْمُنزِلُونَ** (69) | هل أنتم من أنزل هذا الماء من السحب، أم نحن الذين أنزلناه؟ |
| **لَوْ نَشَاءُ جَعَلْنَاهُ أُجَاجًا فَلَوْلَا تَشْكُرُونَ** (70) | لو شئنا لجعلنا هذا الماء مالحًا غير صالح للشرب، فلماذا لا تشكرون الله؟ |
| **أَفَرَأَيْتُمُ النَّارَ الَّتِي تُورُونَ** (71) | هل تأملتم في النار التي تشعلونها من الأشجار؟ |
| **أَأَنتُمْ أَنشَأْتُمْ شَجَرَتَهَا أَمْ نَحْنُ الْمُنشِئُونَ** (72) | هل أنتم من أنشأ شجرتها التي توقدون منها النار، أم نحن الذين خلقناها؟ |
| **نَحْنُ جَعَلْنَاهَا تَذْكِرَةً وَمَتَاعًا لِّلْمُقْوِينَ** (73) | جعلنا هذه النار تذكرة لمن يعتبرون ومتاعًا للمسافرين المحتاجين للنار للتدفئة والطبخ. |
| **فَسَبِّحْ بِاسْمِ رَبِّكَ الْعَظِيمِ** (74) | فمجِّد ربك واذكره دائمًا، فهو العظيم في قدرته. |
| **فَلَا أُقْسِمُ بِمَوَاقِعِ النُّجُومِ** (75) | أقسم بمواقع النجوم في السماء، وهي قسم عظيم. |
| **وَإِنَّهُ لَقَسَمٌ لَّوْ تَعْلَمُونَ عَظِيمٌ** (76) | وإن هذا القسم عظيم لو كنتم تعلمون قدره. |
| **إِنَّهُ لَقُرْآنٌ كَرِيمٌ** (77) | إن هذا القرآن هو كلام كريم ذو قيمة عظيمة. |
| **فِي كِتَابٍ مَّكْنُونٍ** (78) | محفوظ في كتاب مخفي عن أعين الناس وهو اللوح المحفوظ. |
| **لَّا يَمَسُّهُ إِلَّا الْمُطَهَّرُونَ** (79) | لا يمس هذا الكتاب إلا الملائكة المطهرون. |
| **تَنزِيلٌ مِّن رَّبِّ الْعَالَمِينَ** (80) | هذا القرآن مُنزل من الله، رب العالمين. |
| **أَفَبِهَذَا الْحَدِيثِ أَنتُم مُّدْهِنُونَ** (81) | هل بهذا القرآن أنتم متهاونون وغير جادين؟ |
| **وَتَجْعَلُونَ رِزْقَكُمْ أَنَّكُمْ تُكَذِّبُونَ** (82) | وتجعلون شُكركم لله هو التكذيب بالقرآن؟ |
| **فَلَوْلَا إِذَا بَلَغَتِ الْحُلْقُومَ** (83) | فلماذا لا تتصرفون حين تصل روح الإنسان إلى حلقومه عند الموت؟ |
| **وَأَنتُمْ حِينَئِذٍ تَنظُرُونَ** (84) | وأنتم تشاهدون الموت أمام أعينكم. |
| **وَنَحْنُ أَقْرَبُ إِلَيْهِ مِنكُمْ وَلَكِن لَّا تُبْصِرُونَ** (85) | ونحن أقرب إلى المحتضر منكم، ولكنكم لا ترون ذلك. |
| **فَلَوْلَا إِن كُنتُمْ غَيْرَ مَدِينِينَ** (86) | إن كنتم غير محاسبين على أعمالكم كما تزعمون. |
| **تَرْجِعُونَهَا إِن كُنتُمْ صَادِقِينَ** (87) | فأرجعوا روح الميت إلى جسده إن كنتم صادقين في نفيكم للبعث. |
| **فَأَمَّا إِن كَانَ مِنَ الْمُقَرَّبِينَ** (88) | أما إذا كان المتوفى من المقربين إلى الله. |
| **فَرَوْحٌ وَرَيْحَانٌ وَجَنَّةُ نَعِيمٍ**
| **وَأَمَّا إِن كَانَ مِنْ أَصْحَابِ الْيَمِينِ** (90) | وإذا كان المتوفى من أصحاب اليمين. |
| **فَسَلَامٌ لَّكَ مِنْ أَصْحَابِ الْيَمِينِ** (91) | فيُقال له: سلام لك من أصحاب اليمين، دليل على دخوله الجنة. |
| **وَأَمَّا إِن كَانَ مِنَ الْمُكَذِّبِينَ الضَّالِّينَ** (92) | وإذا كان من المكذبين بالحق الضالين عن طريق الله. |
| **فَنُزُلٌ مِّنْ حَمِيمٍ** (93) | فاستقباله في النار سيكون بماء حار يغلي. |
